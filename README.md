# Hassan - Complete Islamic Hub

A comprehensive Islamic application featuring Quran, Hadith, Prayer Times, Qibla Direction, and AI Islamic Scholar.

## Features

- **Quran**: Read and listen to Quran with multiple reciters
- **Hadith**: Search and browse authentic Hadith collections
- **Prayer Times**: Accurate prayer times based on location
- **Qibla Compass**: Find Qibla direction with device orientation
- **Digital Tasbih**: Digital dhikr counter
- **Islamic Calendar**: Hijri calendar integration
- **AI Islamic Scholar**: Chatbot for Islamic questions (English & Kiswahili)
- **99 Names of Allah**: Learn the beautiful names of Allah

## Technologies Used

- HTML5, CSS3, JavaScript
- Bootstrap 5.3.2
- Font Awesome 6.5.1
- Google Fonts (Amiri, Poppins)
- Multiple Islamic APIs (Quran, Hadith, Prayer Times)

## APIs Integrated

- Quran API (Al-Quran Cloud)
- Hadith API (HadithAPI.com)
- Prayer Times API (Aladhan)
- Hijri Calendar API
- Groq AI API for Islamic Scholar

## Setup

1. Clone this repository
2. Open `index.html` in a web browser
3. No additional setup required - all APIs are integrated

## Usage

1. Select your preferred language (English or Kiswahili)
2. Complete the quick setup with your Islamic background
3. Explore different features using the bottom navigation
4. Use the AI Islamic Scholar for any Islamic questions
